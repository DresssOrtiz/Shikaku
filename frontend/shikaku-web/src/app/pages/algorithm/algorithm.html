El Algoritmo

Backtracking Inteligente y Control de Estado

Búsqueda en Profundidad

El núcleo de nuestro solucionador en Python es un algoritmo recursivo de Backtracking. Imagina un laberinto gigante: el algoritmo avanza tomando decisiones (colocando un rectángulo). Si en algún punto detecta que el camino no tiene salida, retrocede, deshace la última decisión en memoria y prueba un camino alternativo.

Pero el Backtracking puro sigue siendo computacionalmente ineficiente. Para dotarlo de inteligencia real y vencer a la explosión combinatoria, implementamos un marco heurístico estricto:

Arquitectura Heurística

1

MRV (Minimum Remaining Values):
La filosofía de "Falla rápido". En lugar de resolver de izquierda a derecha, el algoritmo elige primero la pista que tiene el menor número de combinaciones posibles. Esto colapsa el árbol rápidamente.

2

LCV (Least Constraining Value):
La filosofía de "Sé amable con tus vecinos". Una vez elegida la pista, evalúa qué rectángulo elimina la menor cantidad de opciones para las demás pistas, maximizando las chances de éxito.

3

Poda Activa (Forward Checking):
La filosofía de "Mira antes de saltar". Si colocar una pieza deja a otra pista sin opciones matemáticas o deja celdas "huérfanas" imposibles de cubrir, el algoritmo cancela esa rama instantáneamente.
