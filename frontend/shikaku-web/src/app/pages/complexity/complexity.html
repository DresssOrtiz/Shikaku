La Matemática detrás del Juego

A primera vista, Shikaku parece un simple ejercicio infantil de dibujar cajas. Sin embargo, desde la rigurosa perspectiva de las ciencias de la computación, Shikaku es formalmente un problema de decisión clasificado como NP-Completo.

Esto significa que no existe ningún algoritmo de tiempo polinomial garantizado para resolverlo. Si encontraras uno, resolverías el mayor misterio de las matemáticas: P vs NP.

¿Por qué es tan difícil para una máquina?

A

Explosión Combinatoria: Un solo número puede representarse con múltiples formas (ej: "12" = 1x12, 2x6, 3x4). Además, cada forma puede anclarse en múltiples posiciones.

B

Dependencia Global: Colocar un rectángulo en una esquina afecta drásticamente las opciones de las demás pistas. Cada colocación altera todo el ecosistema del tablero.

Por qué la Fuerza Bruta Fracasa

Probar cada posible rectángulo para cada pista secuencialmente generaría un árbol de búsqueda de proporciones cósmicas en segundos.

Por eso utilizamos Backtracking Inteligente, potenciado con Heurísticas diseñadas para podar ramas enteras del árbol antes de visitarlas.
